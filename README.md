# Python Practice Repository

This repository contains beginner-friendly Python practice programs organized by topic and day. It is intended as a learning workspace for building core programming skills through small, focused scripts.

## Repository Structure

- `day1/` - Introductory input/output practice (for example, a band name generator).
- `day2/` - Basic arithmetic and user input exercises (for example, tip calculator).
- `day4/` - Lists, loops, conditionals, and mini-projects such as:
  - Rock-Paper-Scissors
  - Finding maximum values in a list
  - FizzBuzz
- `day5/` - Password generator and early word-guessing game logic.
- `day6/` - Additional random/word-related practice scripts.
- `Practice_prgrams/` - Algorithm-focused problems, including:
  - Two Sum
  - Best Time to Buy and Sell Stock (single transaction)
  - Buy/Sell for total profit over multiple transactions
- `data/` - Data files used by scripts (`orders.csv`).

## How to Run

1. Ensure Python 3 is installed.
2. Run any script directly from the repository root:

```bash
python day1/main.py
python day4/project.py
python Practice_prgrams/sum_of_two.py
```

## Purpose

The goal of this repository is to practice Python fundamentals, problem solving, and coding logic through hands-on examples.
